# ask.veyza

AI-powered website scanner and chatbot builder for small businesses.

## Features
- Full site audit (15+ checks)
- Local SEO scoring
- Competitor benchmarking
- AI chatbot builder
- Lead capture
- Social media tools
- Events & specials board

## Setup

```bash
npm install
npm start
```

Open http://localhost:5500

## Environment Variables

Copy `.env.example` to `.env` and fill in your values.

## Stack
- Node.js + Express
- Vanilla HTML/CSS/JS frontend
- No database required to start
